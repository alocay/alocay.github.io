Gallery Layout — Grid vs Carousel

How should the Artwork and Photography sections display images?

Artwork
Click any image → lightbox opens

A — Wrap Grid (current)

All images visible at once in a tiled grid. Scroll down to see more. Click any to open full lightbox. Simple, scannable, no interaction needed to see the collection.

Pros

  • Everything visible upfront
  • No extra interaction needed

Cons

  • Can feel dense with many images

B — Horizontal Carousel

Images in a horizontal scroll strip with prev/next arrows. One focal image at a time, others partially visible. Clicking opens the lightbox. More dramatic, less overwhelming with large collections.

Pros

  • Elegant with large image sets
  • Each image gets more focus

Cons

  • Requires interaction to browse
